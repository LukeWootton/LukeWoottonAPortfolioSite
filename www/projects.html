Projects:
Project 1:
In 2019, I was tasked with texturing and adding lighting to pre-made models, I used my skills in substance painter to create high-fidelity textures to help build a convincing environment. Lighting was applied through unreal engine in a number of places to allow for certain effects such as flickering lights.
Project 2:
For this project, I was required to model and texture a vehicle for a racing game. My skills for this project included meticulously building models in Maya to a high standard including both a low-poly version and a high-poly version for rendering. The models were textured using substance painter to fit a given specification.
Project 3:
This is my latest project from Q3 2020, I was given a brief to sculpt different chess pieces to be applied to an online 3D chess game. I used my Mudbox skills to carefully sculpt each piece consistently to a fine detail to be exported into Substance painter so that I could apply the textures provided to me.
